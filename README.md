# Togo Physical Training

ラグビーを中心にしたジュニア選手のフィジカル育成プロジェクト。

## 目的

- 測定結果を継続保存する
- 年齢別目標を管理する
- 強みと課題を見える化する
- トレーニング計画に反映する

## 選手

- Togo
- Sport: Rugby

## 長期目標

- 100m: 10.5
- Vertical Jump: 100cm
- Standing Long Jump: 350cm
