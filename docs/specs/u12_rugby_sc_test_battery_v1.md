# U12 ラグビー S&C テストバッテリー v1.0

（U12 Rugby Strength & Conditioning Test Battery v1.0）

## 1. 概要

本仕様書は、U12ラグビー選手のフィジカル能力を測定・分析するための
テストバッテリーを定義する。

本システムの主目的は以下である。

-   成長トレンドの可視化
-   強み / 弱みの分析
-   トレーニング方針決定
-   長期的なアスリート育成

本ツールは以下の条件を前提として設計されている。

-   個人運用（まずは1人の選手）
-   スマートフォン動画による計測
-   特殊測定機器を使わない
-   継続測定による成長分析

------------------------------------------------------------------------

# 2. 設計思想

## 2.1 単発の能力より成長

このツールは

「今日どれだけ速いか」

ではなく

「どれだけ成長しているか」

を見るためのシステムである。

------------------------------------------------------------------------

## 2.2 最小装備で測定

必要な道具

-   スマートフォン（動画）
-   メジャー
-   コーン
-   ストップウォッチ

------------------------------------------------------------------------

## 2.3 レイヤー構造

本システムは3レイヤー構造で設計されている。

  レイヤー                     目的
  ---------------------------- ----------------
  Performance Analytics        競技能力分析
  Medical / Injury Screening   怪我リスク評価
  Growth Context               成長状況の解釈

※スコア化するのは Performance Layer のみ。

------------------------------------------------------------------------
