# スコアリングモデル仕様（Scoring Model）

## 概要

本プロジェクトでは、フィジカル能力を比較可能にするために **World
Progress Model** を採用する。

このモデルでは、世界トップレベルを基準として
能力を0〜100のスケールで評価する。

------------------------------------------------------------------------

# スコアスケール

  スコア   意味
  -------- -------------------
  0        floor（最低基準）
  50       competitive
  75       elite
  100      world elite

このスケールにより、現在の能力が
世界レベルと比較してどの位置にあるかを直感的に理解できる。

------------------------------------------------------------------------

# radar_score

各テスト結果は次の式で正規化される。

radar_score = 100 × (value - floor_anchor) / (world_elite_p50 -
floor_anchor)

制限

score \< 0 → 0\
score \> 100 → 100

------------------------------------------------------------------------

# Domain Score

テストは能力ドメインごとにまとめられる。

例

Speed Power Elastic COD Upper Endurance

Domain Score は 各ドメインの radar_score の平均で計算される。

------------------------------------------------------------------------

# Rugby Physical Score

総合スコアは以下で計算する。

rugby_physical_score = 平均(domain_scores)

------------------------------------------------------------------------
