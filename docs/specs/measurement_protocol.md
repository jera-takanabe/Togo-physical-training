# 測定プロトコル（Measurement Protocol）

本ドキュメントはフィジカル測定の手順を統一し、
測定誤差を最小化することを目的とする。

------------------------------------------------------------------------

# スプリント測定

## カメラ設置

-   カメラは走路の真横
-   走路中央付近
-   高さ 約1m

カメラが斜めになると計測誤差が発生する。

------------------------------------------------------------------------

## スタート判定

スタートは

「後足が地面を離れた瞬間」

とする。

------------------------------------------------------------------------

## ライン通過判定

タイムは

「胸がラインを通過した瞬間」

とする。

足や腕では判定しない。

------------------------------------------------------------------------

# Fly測定ルール

Fly測定では助走距離を固定する。

助走

20m

測定区間

10m

助走が短すぎると トップスピードではなく加速途中を測ってしまう。

------------------------------------------------------------------------

# ジャンプ測定

CMJ

-   手を腰に固定
-   反動あり

SJ

-   膝を曲げた状態からスタート
-   反動なし
