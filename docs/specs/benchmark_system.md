# Benchmark System v1（ベンチマークシステム仕様）

## 1. 目的

本ドキュメントは、本プロジェクトで使用する フィジカル能力評価の
**ベンチマーク体系** を定義する。

ジュニアアスリートの評価では、
成人トップアスリートと同一基準で評価すると 誤解を生む可能性がある。

そのため本システムでは

**段階的ベンチマークモデル**

を採用する。

------------------------------------------------------------------------

# 2. ベンチマークレベル

評価基準は次の4段階で構成する。

  レベル        意味
  ------------- ------------
  Youth         年齢平均
  Competitive   競技レベル
  Elite         エリート
  World         世界トップ

------------------------------------------------------------------------

# 3. スコア対応

World Progress Model との関係

  Level         Score
  ------------- -------
  Youth         30
  Competitive   50
  Elite         75
  World         100

------------------------------------------------------------------------

# 4. benchmark_values.csv

ベンチマークは以下のデータとして管理する。

例

test,floor_anchor,competitive,elite,world_elite_p50

20m_sprint,4.2,3.6,3.3,3.1 vertical_jump,25,40,50,60

------------------------------------------------------------------------

# 5. Youth Benchmark

ジュニア評価では

world基準ではなく

**youth benchmark**

を補助指標として使用する。

------------------------------------------------------------------------

# 6. 解釈

例

20m sprint

  time   interpretation
  ------ ----------------
  4.2    floor
  3.6    competitive
  3.3    elite
  3.1    world

------------------------------------------------------------------------

# 7. 注意

ジュニア選手では

-   成長スパート
-   身長変化
-   技術習得

により数値が急激に変化する。

そのため

**単発の測定値よりトレンドを重視する。**
