# Sprint 10m / 30m Protocol

## 目的
加速能力を把握する。

## 手順
- 同じ路面・同じシューズ条件で実施
- スタート姿勢を固定する
- Kinovea で後足離地をスタート判定とする
- 5m, 10m, 20m, 30m を split 計測する
- 2〜3本実施し、ベストと平均を残す
